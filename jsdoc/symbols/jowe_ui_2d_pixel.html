Method Detail
draw(items, angle, left, top, colors, zoom)
Draw a pixel map.
The function generates a true image from the heightmap. Each item becomes a pixel.
Even if it's possible to zoom in, it's not recommended as it will produce blurred map.
Parameters:
{array} items
Heightmap to display
{number} angle
To rotate the map (in degree).
{number} left
Left offset from which starting display
{number} top
Top offset from which starting display
{array} colors
Array of colors associated to heightmap.
{number} zoom Optional, Default: 1
Zoom factor (from 0.1 to 4).
