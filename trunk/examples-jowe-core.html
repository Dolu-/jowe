This page explains how to use the HeightMap object in the "jowe-core.js" file. You can give it a try at the bottom of the page.

Initialization:

Calling all methods individually :

  // Create an object with default values :
  // pitch = 8, ratio = 3.1, width = 127, height = 127
  var oMap = new HeightMap();

Method 1:

  // Set the new size.

  // Process the random height map.

  // Shortcut to generate a map (direct calls to "setSide", "makeMap", "smooth" and "crop").

Working with pitch and ratio parameters:

The map is an array which contains values (integer) between 0 and {pitch}.
You can set an higher pitch, but it will more effective on bigger maps.

  var oMap = new HeightMap(99);
  oMap.doMap(14, 34);
