HeightMap(opt_pitch, opt_ratio, opt_width, opt_height)
Creates a new random height map generator object.
Parameters:
{number} opt_pitch Optional, Default: 8
Maximum elevation to be used. Items in the map will have a value between 0 and opt_pitch.
{number} opt_ratio Optional, Default: 3.1
Ratio to be used.
{number} opt_width Optional, Default: 127
Width of the map to be generated.
{number} opt_height Optional, Default: 127
Height of the map to be generated.

How to use the HeightMap object?
Field Detail
<private> <constant> {number} CROP_SIZE
Borders of the working height map to exclude from final result, because there are not processed in the smooth function.
{Array} item
Result array with the world map.
<private> <constant> {number} MAX_SIDE
Set maximum size for the side value of the working square.
<private> <constant> {number} MIN_SIDE
Set minimum size for the side value of the working square.
<private> {number} p_pitch
Maximum elevation for current map [0 to p_pitch].
You will have to adjust the color managment in "jowe-ui" to fit your elevation.
<private> {number} p_ratio
Indicates how much height difference between 2 points we can have. Only used in "make" method.
By the way, combined with "p_pitch" (previous property), it allows to obtain very different types of map.
For now 3.1 is around the minimum value to use, below that you could obtain strange map (unmanaged cell display). If you put an higher value, your map will look flattened.
<private> {number} p_side
Size of the current map [0 to p_side], [0 to p_side].
Always have to be (N^2)+1 x (N^2)+1 ("diamond square" algorithm. Better looking results with squares). We only need to store it once (p_side = width = height).

doMap(p_width, p_height, bInitialize)
Build a map (size is 0-based).
doMap(5, 10) will return a map with dimension [0 .. 4][0 .. 9] but as we need 2 points to make a cell we'll have 4x9 cells (= 36 true cells displayed). 
var oMap = new HeightMap(99, 0.54);
oMap.doMap(14, 34);
Parameters:
{number} p_width
{number} p_height
{boolean} bInitialize Optional, Default: true
Tell to the function to goes through an initialization of the map array. Then will call "SetSide", "initialize" and "fillcorners" with default values.

makeMap()
This is a shortcut to call function make() with default parameters.
All necessary variables should have been initialized.
The "item" array will be set to "-1" if not already done.
<private> {number} p_addDelta(avg, delta)
Return a value added with a random delta.
If the delta is more than 0, it adds to "avg" a random value between "-delta" and "+delta".
In fact it sets the height between two points. The closer there are, the lower the delta will be. The function ensure that the result is positive and less than the pitch (max height).
Parameters:
{number} avg
{number} delta
Returns:
{number}
